php-router by robap

php routing class

The Router class is designed to map url's to controller::action methods in your MVC framework. It is not in itself a framework. To use it, you will need to create a index/bootstrap file and configure apache to route requests there. Included is a sample.htaccess file which you can copy into your apache public directory along with your index/bootstrap file.

Dependencies

>= php 5.2.x

Install

1. Copy Router.php to some place accessible by your framework and include it. 2. Create your routes for your controller/actions (see the 'Adding Routes' section below). 3. Create your index.php file and call any desired Router methods. 4. Tell apache to route requests through index.php (see sample.htaccess). 5. Call Router::mapUrlRequest at the end of your index.php file.
